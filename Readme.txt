This project has two files:

1) Recipe_ui.ipynb
This file contains a partially working UI for recipe. 
This program takes voice as an input and returns attributes of the recipe.
Click on small robot to start.
Click on ask me to give input to the programme.


2) recipe_parser.ipynb 
This file scrapes the complete recipe from allrecipe.com and returns all the required results.
Kindly run all the cells to get all the necessary features of the recipe.
The code also has an option to transform a recipe to vegetarian.
The last cell is for all the question answering
